# AI Agent Automation System

A multi-agent automation system using Claude, Gemini, and local LLMs.

## 🚀 Overview
This project explores automation using multiple AI agents working together.

## ⚙️ Features
- Multi-agent system
- Task automation
- Hybrid AI (API + local models)

## 🧠 LLM Integration
Supports:
- Claude (Anthropic API)
- Gemini
- Local models (Ollama)

Currently using a mock LLM for testing.
